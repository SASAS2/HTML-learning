## 画盒子

### 1. HTML结构

使用 `div` 标签创建不同颜色的盒子。

```css
        .blue {
            width: 500px;
            height: 50px;
            background-color: rgb(176, 216, 249);
        }
        .pink {
            width: 500px;
            height: 50px;
            background-color: rgb(252, 208, 215);
        }
        .white {
            width: 500px;
            height: 50px;
            background-color: white;
        }
```

### 2. CSS样式

使用类选择器为每个盒子设置宽度、高度和背景颜色。

- `.blue` 类设置背景颜色为蓝色。
- `.pink` 类设置背景颜色为粉色。
- `.white` 类设置背景颜色为白色。
